# Golden Analytics POC

A lightweight proof-of-concept dashboard for exploring government spending data.

This repo includes:
- A React + Vite frontend with interactive charts, filters, and a natural-language “ask a question” panel
- A .NET minimal API backend for vehicle management and future logging
- AI interaction logging for governance, saved locally and sent to `/api/logs`
- A sample dataset and clean, folder-based project structure

The goal is to show the core product flow quickly: data exploration, simple AI query handling, and audit logging.
